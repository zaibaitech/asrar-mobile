# Arabic Keyboard - Quick Usage Guide

## How to Use

### 1. Opening the Keyboard
- Locate the "Show Keyboard" button next to the Arabic Name field
- Tap the button to open the keyboard
- The keyboard will slide up from the bottom with a smooth animation

### 2. Typing Arabic Letters
- Default view shows all 28 Arabic letters in a 5-row layout
- Tap any letter to insert it at the cursor position
- Letters are arranged right-to-left as in standard Arabic keyboards

### 3. Adding Diacritics (Tashkeel)
- Tap the "التشكيل" button at the top to switch to diacritics
- Choose from 8 common diacritical marks:
  - Fatha (َ)
  - Tanween Fath (ً)
  - Kasra (ِ)
  - Tanween Kasr (ٍ)
  - Damma (ُ)
  - Tanween Damm (ٌ)
  - Sukun (ْ)
  - Shadda (ّ)
- Tap "الحروف" to return to letters

### 4. Editing Text
- **Backspace (⌫)**: Deletes the character before cursor
- **Space (مسافة)**: Inserts a space
- **Cursor Position**: Tap in the input field to move cursor
- Text inserts at cursor position, not always at the end

### 5. Closing the Keyboard
Two ways to close:
- Tap the "إغلاق" (Close) button
- Tap anywhere outside the keyboard

## Tips

✅ **DO:**
- Use the physical device keyboard if available for faster typing
- Switch between letters and diacritics as needed
- Use backspace to correct mistakes
- Tap in the input field to position cursor

❌ **DON'T:**
- Don't rely solely on visual appearance - verify spelling
- Don't forget to add diacritics for proper pronunciation
- Don't close keyboard before reviewing your input

## Troubleshooting

**Problem: Keyboard doesn't appear**
- Solution: Ensure "Show Keyboard" button is tapped
- Check if modal overlay is visible

**Problem: Text appears in wrong position**
- Solution: Tap in input field to set cursor position
- Characters insert where cursor is located

**Problem: Can't find a letter**
- Solution: All 28 letters are visible in letters view
- Scroll if screen is small
- Check you're not in diacritics view

**Problem: Keyboard covers input**
- Solution: Input field is above keyboard
- Scroll form if needed
- Keyboard is semi-transparent overlay
