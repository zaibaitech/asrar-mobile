## Letter Organization

### Complete Arabic Alphabet (28 Letters)

Organized in QWERTY-style layout for familiarity:

| Row | Letters | Count |
|-----|---------|-------|
| 1   | ض ص ث ق ف غ ع | 7 |
| 2   | ه خ ح ج د ش س | 7 |
| 3   | ظ ط ذ ز ر و ي | 7 |
| 4   | ة ى ء ؤ ئ ا ب | 7 |
| 5   | ن م ك ل ت | 5 |

**Total: 28 letters** ✓

### Alphabetical Order Reference

Standard Arabic alphabetical order (Abjad):

```
ا ب ت ث ج ح خ د ذ ر ز س ش ص ض ط ظ ع غ ف ق ك ل م ن ه و ي
```

Additional forms:
```
ء ؤ ئ ة ى
```

## Diacritics (Tashkeel)

### 8 Common Diacritical Marks

| Symbol | Name (Arabic) | Name (English) | Unicode | Function |
|--------|---------------|----------------|---------|----------|
| َ | فتحة | Fatha | U+064E | Short 'a' sound |
| ً | تنوين فتح | Tanween Fath | U+064B | '-an' ending |
| ِ | كسرة | Kasra | U+0650 | Short 'i' sound |
| ٍ | تنوين كسر | Tanween Kasr | U+064D | '-in' ending |
| ُ | ضمة | Damma | U+064F | Short 'u' sound |
| ٌ | تنوين ضم | Tanween Damm | U+064C | '-un' ending |
| ْ | سكون | Sukun | U+0652 | No vowel/silence |
| ّ | شدة | Shadda | U+0651 | Doubling/emphasis |

## Key Functions

### Bottom Row Controls

```
┌─────────────┬─────────────┬─────────────┐
│  Backspace  │    Space    │    Close    │
│     ⌫       │   مسافة     │   إغلاق     │
│   (Red)     │   (Gray)    │   (Gray)    │
└─────────────┴─────────────┴─────────────┘
```

**Backspace (⌫)**
- Color: Red (#EF4444)
- Function: Delete character before cursor
- Flex: 1 (25% width)

**Space Bar (مسافة)**
- Color: Gray (#4B5563)
- Function: Insert space character
- Flex: 2 (50% width)

**Close (إغلاق)**
- Color: Gray (#6B7280)
- Function: Dismiss keyboard
- Flex: 1 (25% width)

## Responsive Behavior

### Screen Size Adaptations

**Small Screens (<375px)**
- Keys: Smaller width
- Font: Same size (readability)
- Scroll: Enabled
- Rows: Maintain 5-row layout

**Medium Screens (375-768px)**
- Keys: Optimal size
- Font: Standard
- Scroll: Minimal
- Rows: Full visibility

**Large Screens (>768px)**
- Keys: Max width ~60px
- Font: Standard
- Scroll: Disabled
- Rows: Centered layout

## Usage Patterns

### Common Input Sequences

**Typing a Name (محمد)**
```
1. Tap: م
2. Tap: ح
3. Tap: م
4. Tap: د
Result: محمد
```

**Adding Diacritics (مُحَمَّد)**
```
1. Type: محمد
2. Toggle to diacritics
3. Position cursor after م
4. Tap: ُ (damma)
5. Position after ح
6. Tap: َ (fatha)
7. Position after second م
8. Tap: َ (fatha)
9. Tap: ّ (shadda)
Result: مُحَمَّد
```
